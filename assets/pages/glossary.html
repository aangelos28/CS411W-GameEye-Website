Glossary

PWA

An acronym for Progressive Web Application; it is a type of application software delivered through the web, built using common web technologies including HTML, CSS, and JavaScript.

REST

An acronym for REpresentational State Transfer; it is a software architectural style that defines a set of constraints to be used for creating web services.

API

An acronym for application programming interface; it is a set of functions and procedures allowing creation of applications that access the features or data of an operating system, application, or other service.

RSS Feed

A web feed that allows users and applications to access updates to websites in a standardized, computer-readable format.

Web Scraping

Data scraping for extracting data from websites.

IGDB

A database of known video games; GameEye will utilize its REST API to populate its database.

Indie Games

Games developed by individuals or smaller teams of people without financial support of larger game publishers.

Hitlist

A list of highly watched video games by users

OIDC Authentication

An authentication protocol based on the OAuth2.0 family of specifications.

HTML

Hypertext Markup Language, it is used as markup for documents meant to be displayed in a web browser.

CSS

Cascading Style Sheets; used to stylize web pages.

JavaScript

Object-oriented language used to create dynamic, interactive effects on web pages.

Python

An interpreted, high-level, general-purpose programming language.

Angular Framework

A platform for building mobile and desktop applications

Jest JavaScript Framework

A testing framework maintained by Facebook Inc. with a focus on simplicity.

JUnit Java Framework

The most popular testing framework for Java.

JSoup Library

A Java library for working with real-world HTML; used for fetching URLs, extracting and manipulating data.

Keras

An open-source neural-network library written in Python.

scikit-learn

An open-source machine learning library for the python programming language.

spaCy

An open-source software library for advanced natural language processing.

MongoDB

A cross-platform document-oriented database program.

AWS

Amazon® subsidiary that provides on-demand cloud computing Platforms and APIs.

IntelliJ IDEA

An IDE developed by JetBrains for the purpose of writing Java and will be used in the back-end development of GameEye.

WebStorm

An IDE developed by JetBrains for the purpose of writing JavaScript and will be used in the front-end development of GameEye.
