Glossary

Angular Framework
- it is a platform for building mobile and desktop applications
API
- an acronym for application programming interface; it is a set of functions and procedures allowing creation of applications that access the features or data of an operating system, application, or other service.
AWS
- Amazon® subsidiary that provides on-demand cloud computing Platforms and APIs
CSS
- Cascading Style Sheets; used to stylize web pages.
Hitlist
- a list of highly watched video games by users
HTML
- Hypertext Markup Language, it is used as markup for documents meant to be displayed in a web browser.
IGDB
- is a database of known video games; GameEye will utilize its REST API to populate its database.
Indie Games
- games developed by individuals or smaller teams of people without financial support of larger game publishers.
IntelliJ Idea
- IDE developed by JetBrains for the purpose of writing Java and will be used in the back-end development of GameEye®.
JavaScript
- Object-oriented language used to create dynamic, interactive effects on webpages.
Jest JavaScript Framework
- testing framework maintained by Facebook Inc with a focus on simplicity
JSoup Library
- Java library for working with real-world HTML; used for fetching URLs, extracting and manipulating data
JUnit Java Framework
- testing framework with importance in the realm of test-driven development
Keras(Python Deep Learning Library)
- it is an open-source neural-network library written in python
MongoDB
- a cross-platform document-oriented database program.
Noise Filtering
- information/news articles is shown that caters to an individual's content preferences.
OIDC Authentication
- OIDC is an authentication protocol, based on the OAuth2.0 family of specifications.
PWS
- an acronym for Progressive Web Application; it is a type of application software delivered through the web, built using common web technologies including HTML, CSS, and JavaScript.
Python
- it is an interpreted, high-level, general-purpose programming language
REST
- an acronym for representational state transfer; it is a software architectural style that defines a set of constraints to be used for creating web services.
RSS Feeds
- web feed that allows users and applications to access updates to websites in a standardized, computer-readable format.
Scikit-learn library
- free software machine learning library for the python programming language.
spaCy Library
- open- source software library for advanced natural language processing
Web Scraping
- data scraping for extracting data from websites.
WebStorm
- IDE developed by JetBrains for the purpose of writing JavaScript and will be used in the front-end development of GameEye.
